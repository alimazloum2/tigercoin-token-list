Open in MetaMask Mobile

Import TGR manually in MetaMask:
  1. Open MetaMask.
  2. Switch network to Base.
  3. On the Tokens screen, tap Import tokens.
  4. Paste contract: 0xa79d3083a7e303Db38e013C54135F178834Ccb1B
  5. MetaMask should fill TGR and 18. If not, enter them manually.
  6. Tap Import.

Manual fallback

  1. Open this page inside the MetaMask mobile browser, not Telegram/Safari.
  2. Use the orange Copy TGR Contract + Show Import Steps button.
  3. If needed, switch MetaMask to Base.
  4. Manual import contract: 0xa79d3083a7e303Db38e013C54135F178834Ccb1B, symbol TGR, decimals 18.

MetaMask Mobile is currently rejecting the automatic wallet_watchAsset add-token request for this Base token on some devices. Manual custom-token import is the deterministic path. The auto-import button is left only for testing.

Transparency and risk status

  • Verified contract: public BaseScan contract link is provided above.

  • Icon status: the hosted icon is valid, but MetaMask manual imports may still show a default letter icon until MetaMask accepts/indexes the token metadata.
  • No guarantee: this page is informational only and is not financial advice. Token value and liquidity can change quickly.

Current trust-building roadmap

  1. Keep verified contract, token list, and icon publicly available.
  2. Increase real TGR/WETH liquidity when funds are available.
  3. Lock meaningful Uniswap v3 LP liquidity and publish the lock proof.
  4. Update wallet/listing submissions with the verified contract, pool, logo, tokenomics, and lock proof.
  5. Grow real holders and organic usage without fake volume or wash trading.
